> AI半导体全产业链（上下游完整版）—— 从原材料到终端应用的深度调研文档

本知识库系统梳理了AI半导体产业链的完整层次结构，覆盖**上游（原材料与设备）**、**中游（芯片设计、晶圆代工、存储芯片、封装测试）**、**下游（算力硬件、基础设施、终端AI、软件配套）**及**细分配套产业链**四大板块，共32个核心概念详情页。
